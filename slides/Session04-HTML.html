Scraping websites¶

How I scrape a page

  1. Look at the source code of a sample page.
  2. Download the page.
  3. Find the thing that you want, and the stuff around that thing.
  4. Write a regular expression that matches what you want.
  5. Write regular expression that actually matches what you want.
  6. Test that it works on one page.
  7. Production!

  2. Download the page.
  3. Find the thing that you want, and the stuff around that thing.
  4. Parse the HTML.
  5. Test that it works on one page.
  6. Production!
